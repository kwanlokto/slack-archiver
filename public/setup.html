How to get a token

  1. Open api.slack.com/apps → Create New App → From scratch. Pick the workspace you own and name it (e.g. "Archiver").
  2. In the new app, left sidebar → OAuth & Permissions.
  3. Scroll to Scopes → User Token Scopes (not Bot Token Scopes). Add:
    • Scroll up to the top of the same page. Click Install to Workspace → Allow.
    • Slack drops you back on the OAuth & Permissions page. Copy the User OAuth Token that now appears at the top — it starts with xoxp-.

    You do not need to configure a Redirect URL — this flow doesn't use one.

    Paste your token

    The token is stored in data/credentials.json on this server, with file mode 0600.
